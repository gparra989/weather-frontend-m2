# 🌦️ weather-frontend-m2

Aplicación de clima desarrollada en **HTML, CSS con un poco de Javascript** que permite consultar (mediante plantillas) el estado actual 
y el pronóstico del tiempo en diferentes ciudades.

---

## 🚀 Características
- Visualización de temperatura, humedad, velocidad del viento y condiciones generales.
- Pronóstico semanal.
- Interfaz responsiva y amigable.

---

## 🛠️ Tecnologías utilizadas
- **Frontend:** HTML, CSS, JavaScript.
- **Extras:** Bootstrap 5.
